# 📡 Antenna Fault Detection using Machine Learning

## Project Overview

This project focuses on detecting faults in antenna systems using Machine Learning techniques. The goal is to build a predictive model that can classify whether an antenna is functioning properly or experiencing faults based on sensor/data inputs.

This helps in:

Reducing manual inspection efforts

Improving system reliability

Enabling predictive maintenance

## Dataset

File: antenna_dataset.csv

Contains antenna performance parameters and fault labels

Includes features such as signal strength, frequency variations, and other technical metrics
